Introdução ao Básico de CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para definir a aparência de documentos HTML ou XML. Ele permite controlar cores, fontes, espaçamentos, layouts e outros aspectos visuais de uma página web. O CSS é essencial para criar designs atraentes e responsivos.

Estrutura Básica de CSS

Um conjunto de regras CSS é composto por um seletor e um bloco de declarações. 

p {
color: red;
font-size: 16px;
}

Neste exemplo: p é o seletor, color e font-size são propriedades, e red / 16px são valores.

Aplicando CSS

Inline

<p style="color: blue;">Texto azul</p>

Interno

<style>
p {
color: green;
}
</style>

Externo

<link rel="stylesheet" href="estilo.css">

Conceitos Fundamentais

Seletores

  • p: Seleciona todos os parágrafos
  • .classe: Seleciona por classe
  • #id: Seleciona por ID

Box Model

  • Content
  • Padding
  • Border
  • Margin
div {
width: 200px;
padding: 10px;
border: 5px solid black;
margin: 20px;
}

Propriedades Comuns

  • color: red;
  • background-color: yellow;
  • font-size: 14px;
  • margin: 10px;
  • padding: 5px;

Layouts Modernos

Flexbox

.container {
display: flex;
justify-content: center;
align-items: center;
}

Grid

.container {
display: grid;
grid-template-columns: 1fr 1fr;
gap: 10px;
}

Conclusão

O CSS é uma ferramenta poderosa para estilizar páginas web. Com ele, é possível criar designs modernos e responsivos.
