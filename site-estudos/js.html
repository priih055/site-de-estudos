O que é JavaScript?

JavaScript é uma linguagem de programação de alto nível, interpretada e amplamente utilizada para criar páginas web interativas. Ela é uma das três tecnologias fundamentais da web, ao lado de HTML e CSS.

Criada em 1995 por Brendan Eich, inicialmente chamada de LiveScript, foi renomeada para JavaScript. Atualmente, é padronizada pela especificação ECMAScript.

Características Principais

  • Multiparadigma (funcional, orientada a objetos e procedural)
  • Tipagem dinâmica
  • Execução no cliente e no servidor
  • Interatividade com o DOM

Exemplo Prático

Console

console.log("Olá, mundo!");

Interatividade

<button onclick="mostrarMensagem()">Clique aqui</button>

<script>
function mostrarMensagem() {
alert("Você clicou no botão!");
}
</script>

Aplicações

  • Front-end (React, Angular)
  • Back-end (Node.js)
  • Mobile (React Native)
  • Jogos e animações

Vantagens e Desvantagens

  • ✔ Fácil de aprender
  • ✔ Muito utilizado
  • ✔ Versátil
  • ❌ Pode ter falhas de segurança
  • ❌ Diferenças entre navegadores

Conclusão

JavaScript é essencial para o desenvolvimento web moderno e uma das linguagens mais utilizadas no mundo.
