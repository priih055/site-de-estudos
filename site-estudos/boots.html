Estrutura HTML com Bootstrap

O Bootstrap é um framework front-end amplamente utilizado para criar sites responsivos e modernos de forma rápida e eficiente. Ele fornece uma estrutura baseada em HTML, CSS e JavaScript, com componentes prontos e um sistema de grid flexível.

Exemplo de Estrutura HTML com Bootstrap

<!DOCTYPE html>
<html lang="pt-br">
<head>

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1">
<title>Exemplo com Bootstrap</title>

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/css/bootstrap.min.css">

</head>
<body>

<nav class="navbar navbar-expand-lg navbar-light bg-light">
<div class="container-fluid">

<a class="navbar-brand" href="#">Navbar</a>

</div>
</nav>

<div class="container mt-5">
<h1 class="text-center">Bem-vindo ao Bootstrap</h1>
<p class="text-center">Exemplo básico</p>
</div>

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/js/bootstrap.bundle.min.js"></script>

</body>
</html>

Explicação do Código

  • Meta Tags: garantem responsividade
  • CSS do Bootstrap: adiciona estilos prontos
  • Navbar: menu responsivo
  • Container: organiza o conteúdo
  • Scripts: ativam funcionalidades

Benefícios do Bootstrap

  • Responsividade automática
  • Componentes prontos
  • Facilidade de uso

Conclusão

Com Bootstrap, você consegue criar sites modernos e responsivos de forma rápida, sem precisar escrever todo o CSS do zero.
