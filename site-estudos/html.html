Introdução ao HTML

HTML Tutorial - W3Schools HTML básico - MDN

HTML (HyperText Markup Language) é a linguagem padrão usada para estruturar e exibir conteúdo na web.

Estrutura Básica de um Documento HTML

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Minha Página</title>
</head>
<body>
<h1>Bem-vindo ao HTML</h1>
<p>Este é um parágrafo de exemplo.</p>
<a href="https://www.exemplo.com">Clique aqui</a>
</body>
</html>

Explicação dos Elementos

  • <!DOCTYPE html>: Define o tipo do documento
  • <html>: Elemento principal
  • <head>: Metadados
  • <body>: Conteúdo visível

Elementos Comuns do HTML

Cabeçalhos

<h1>Título Principal</h1>
<h2>Subtítulo</h2>

Parágrafos

<p>Este é um parágrafo simples.</p>

Links

<a href="https://www.exemplo.com">Visite nosso site</a>

Imagens

<img src="imagem.jpg" alt="Descrição da imagem">

Listas

<ul>
<li>Item 1</li>
<li>Item 2</li>
</ul>

Boas Práticas

  • Sempre feche as tags
  • Use alt em imagens
  • Organize com cabeçalhos
